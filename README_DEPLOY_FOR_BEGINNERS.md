# Beginner Deployment Notes

1. Open your GitHub repository.
2. Choose **Add file → Upload files**.
3. Upload the extracted contents of this folder, not the zip file.
4. Commit directly to the `main` branch.
5. In Streamlit Cloud, deploy `app.py`.
6. If the page does not refresh, use **Manage app → Reboot app** and open the site with `Ctrl + F5`.
