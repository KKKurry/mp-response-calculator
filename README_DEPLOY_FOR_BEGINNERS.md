# Beginner deployment notes

1. Unzip this folder.
2. Open your GitHub repository.
3. Choose **Add file → Upload files**.
4. Upload the visible files and folders from this folder.
5. Choose **Commit directly to the main branch**.
6. Click **Commit changes**.
7. Open Streamlit Cloud and click **Manage app → Reboot app**.
8. After 1–3 minutes, press **Ctrl + F5** in the browser.

You do not need to upload hidden `.streamlit` if Windows blocks it.
